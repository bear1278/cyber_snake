# Cyber Snake

[![Go Version](https://img.shields.io/badge/Go-1.21+-00ADD8?style=flat&logo=go)](https://golang.org)
[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

Классическая игра змейка с добавлением препятствий в виде вирусов. При столкновении с вирусом игра заканчивается. При увеличении счета на 5 уровень увеличивается и на игровом поле появляется еще один вирус.

## 🔧 Установка

### Из исходного кода

```bash
# Клонирование репозитория
git clone https://github.com/bear1278/cyber_snake.git
cd cyber_snake

# Сборка проекта
go build .

# Запуск
./cyber_snake
```

# Управление

Управление змейкой с помощью стрелок и WASD
Выход из игры Q/Esc
R - перезапуск
